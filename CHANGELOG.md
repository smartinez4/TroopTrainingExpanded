## Changelog

## 1.1.4
Added a toggle in settings.json to enable or disable horses in training fights
Added French and Italian localizations 

## 1.1.3
When a tournament is in progress the option to do training fights will be disabled, same as practice fights
Added German localization by: EagleFour
Added Turkish and Polish localization 

## 1.1.2
If only companions are selected they will spawn as enemies
Added icon in town menu
Disable training fight menu if hero is wounded

## 1.1.1
Added translation template for localization
Added spanish localization, more will come

## 1.1.0
- Added team battles: companions can now be selected to join a practice fight as allies
- Companions will also use horses when appropriate, same as the player

## 1.0.2
- If cavalry units are selected they will equip their horse and the player will too
- Added JSON configuration file logic to allow users to customize settings
- Added MaxTrainingTroops setting to config file

## 1.0.1

- Added troop wounding system: defeated training opponents become wounded and retain status through promotion.
- Added wounded troop restriction: wounded units cannot be selected for training fights.
- Fixed health synchronization: player health now correctly carries over after arena training fights.
- Improved internal logic for promotion handling, roster updates, and mission spawning.
